Built for live lessons

Structure participation without slowing the room down.

Random Student Selector links the classroom routines teachers already run: who is present, who gets called, how long they have to think, what happened, and who may need follow-up after class.

Classroom flow

From roster to recap in one lesson rhythm.

01

Pick the active class

Load the current roster and keep absent students out of the live selection pool.

02

Set the response window

Choose a preset timer and use the optional reel effect to make the selection visible.

03

Capture the outcome

Record A*, A, B, C, No Grade, or Absent as soon as the response finishes.

04

Review the session

See remaining students, participation results, absences, and grade-band counts.

Teacher-facing interface

A compact control dock with the whole lesson state in view.

The interface is designed for repeated use during class: large controls, clear counts, immediate attendance access, optional sound, and a single high-confidence action to start the next selection.

Feature set

Everything needed for fair, low-friction participation.

Attendance-aware roster

Roll call removes absent students from the active session so selection stays credible.

Visible wait time

Timer presets create a response window that gives students a reason to prepare.

Quick formative outcomes

Record contribution quality in the moment without turning discussion into data entry.

Session memory

Chosen and absent students stay excluded while the app remains open.

Classroom-ready audio

Optional intro, closing, slot, timer, and rating cues can be toggled from the dock.

Windows-aware layout

High-DPI and multi-monitor handling keep the app usable on real classroom setups.

Local setup

Run from source or package with PyInstaller.

The app uses Python, Tkinter, ttkbootstrap, and optional pygame audio. Class data stays local in an ignored CSV file.

Install pip install ttkbootstrap pygame
Run python studentselector.py
Package pyinstaller --noconfirm studentselector.spec
